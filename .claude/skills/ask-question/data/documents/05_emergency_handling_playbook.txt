差旅异常与应急处理指南

定位：本文件是操作型知识，用于指导助手在航班延误、取消、误机、酒店问题、证件遗失等场景下给出步骤化建议。报销可行性以学校制度和财务审核为准。

一、航班延误

1. 先确认信息：向航空公司、机场或官方 App 确认延误原因和预计起飞时间；判断是否影响会议签到、报告、调研或返程。
2. 再调整行程：延误较短时通知相关方并继续等待；延误较长时比较改签其他航班、高铁、顺延行程的可行性；重要会议或任务不要只等航空公司安排，要主动评估替代交通。
3. 保留凭证：延误证明或官方通知、改签/退票/补票费用凭证、由延误产生的必要住宿和交通凭证、与会议方或任务方沟通的记录。

二、航班取消

1. 立即确认取消原因和航空公司方案。
2. 优先选择能保证任务完成的替代交通。
3. 若改签后无法完成任务，应提醒用户联系导师、项目负责人、会议方或单位负责人。
4. 保留取消通知、退改签凭证和新交通订单。
5. 回答中要区分航空公司承担费用和学校报销判断。

三、误机或误车

1. 先到柜台或官方渠道确认是否可改签。
2. 判断原因：个人原因导致的损失通常风险较高；不可抗力或公务安排导致的变化，需要保留证明和说明。
3. 如果任务紧急，优先重新规划可达路线。
4. 提醒用户保留原订单、改签订单、支付凭证和情况说明。

四、行李延误或丢失

1. 在机场行李查询处登记并获取报告编号。
2. 保留行李牌、登机牌、报告单、沟通记录。
3. 如影响公务任务，可购买必要用品，但要控制合理范围并保留发票。
4. 如果携带科研材料、设备或会议资料，应及时通知相关方并准备替代方案。

五、酒店到店无房或房间异常

1. 到店无房：要求酒店提供同等级或合理替代方案；保留酒店确认无房的证据和沟通记录；如需换酒店，优先选择任务地点附近、价格合理、可开发票的酒店。
2. 房间卫生、设施或安全问题：先拍照并联系前台换房；安全隐患严重时，应优先更换住宿；保留沟通记录、照片、额外交通或住宿凭证。

六、证件遗失

1. 身份证遗失：到当地派出所或车站/机场相关窗口咨询临时身份证明；保留办理证明和相关记录。
2. 护照或港澳台证件遗失：境外应尽快报警并联系中国驻当地使领馆；保留报案证明、补办材料、使领馆证明；同时通知学校外事或项目负责人。

七、人身安全和突发疾病

1. 先保证安全和就医，不要为了保留报销凭证延误处理。
2. 保存医院诊断、收费票据、保险材料。
3. 医疗费用是否属于差旅报销范围要查制度，不能默认可报。
4. 如影响行程，应及时通知会议方、导师或项目负责人。

八、助手回答模板

1. 先给立即行动步骤。
2. 再给替代行程选择。
3. 再给应保留的凭证。
4. 最后提醒报销风险和需要补充说明。
